## STEP 3: Upload Files to GitHub (Easiest Method - No Git Needed!)

1. **Go back to your GitHub repository page**:

2. **Click the button**: "uploading an existing file" (or "Add file" ‚Üí "Upload files")

3. **Drag and drop ALL files** from your folder:

   - Drag `styles.css` into the box
   - Drag `script.js` into the box

## STEP 4: Enable GitHub Pages (Make Website Live)

1. **On your repository page**, click **"Settings"** (top menu, far right)

3. **Under "Source"**:

   - **Folder**: Select **"/ (root)"**
   - Click **"Save"** button

4. **You'll see a message**: "Your site is live at https://triplepower.github.io"

## üîß Troubleshooting

### Problem: "No results for main//(root)"
**Solution**: 
- Make sure you uploaded files first (Step 3)
- Try selecting "master" instead of "main" in the branch dropdown

### Problem: Website shows 404 error
**Solution**: 
- Wait 2-3 minutes (GitHub needs time to build)
- Refresh the page
- Check that files were uploaded correctly

### Problem: Can't find my folder
**Solution**: 
- Open File Explorer
- Go to: `C:\Users\vineshh\snake-water-gun-game\Triplepower.github.io`
- Or search for "Triplepower.github.io" in File Explorer

### Problem: Files won't upload
**Solution**: 
- Make sure you're dragging files, not the folder
- Upload one file at a time if needed
- Make sure you're on the repository page (not GitHub homepage)
